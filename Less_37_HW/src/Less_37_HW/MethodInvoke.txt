******* Object invoke(Object obj, Object... args) *******

public Object invoke(Object obj, Object... args) throws IllegalAccessException,
                                                        IllegalArgumentException,
                                                        InvocationTargetException


Параметры:
obj - объект, из которого вызывается базовый метод;
args - аргументы, используемые для вызова метода;

Возвращает:
результат отправки метода, представленного
этим объектом, в obj с параметрами args.

Метод *.invoke() класса Method, вызывает базовый метод, представленный текущим
объектом Method, для указанного объекта с указанными параметрами. Отдельные
параметры автоматически разворачиваются, чтобы соответствовать примитивным
формальным параметрам, а примитивные и ссылочные параметры при необходимости
подвергаются преобразованиям вызова метода.

Если базовый метод является статическим, указанный аргумент obj игнорируется.
Он может быть нулевым.

Если количество формальных параметров, требуемых базовым методом, равно 0,
предоставленный массив args может иметь длину 0 или null.

Если базовый метод является методом экземпляра, он вызывается с использованием
поиска динамического метода, как описано в спецификации языка Java, второе издание,
раздел 15.12.4.4; в частности, произойдет переопределение на основе типа среды
выполнения целевого объекта.

Если базовый метод является статическим, класс, объявивший метод, инициализируется,
если он еще не был инициализирован.

Если метод завершается нормально, возвращаемое им значение возвращается инициатору
вызова; если значение имеет примитивный тип, оно сначала соответствующим образом
упаковывается в объект. Однако, если значение имеет тип массива примитивного типа,
элементы массива не оборачиваются в объекты; другими словами, возвращается массив
примитивного типа.

Если базовый тип возвращаемого значения метода имеет значение void, вызов возвращает
значение null.


Исключения:
- IllegalAccessException — если этот объект метода обеспечивает контроль доступа к языку
  Java, а базовый метод недоступен.

- IllegalArgumentException выбрасывается:
    - если метод является методом экземпляра, а указанный аргумент объекта не является
      экземпляром класса или интерфейса, объявляющего базовый метод (или его подкласс
      или его реализацию);
    - если количество фактических и формальных параметров различается;
    - если происходит сбой преобразования преобразования для примитивных аргументов;
    - если после возможной распаковки значение параметра не может быть преобразовано
      в соответствующий формальный тип параметра путем преобразования вызова метода.

- InvocationTargetException — если базовый метод вызывает исключение.

- NullPointerException — если указанный объект имеет значение null и метод является
  методом экземпляра.

- ExceptionInInitializerError - если инициализация, спровоцированная этим методом, не удалась.
