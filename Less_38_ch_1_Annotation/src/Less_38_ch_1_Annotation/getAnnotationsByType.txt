******* *.getAnnotationsByType() *******

default <T extends Annotation> T[] getAnnotationsByType(Class<T> annotationClass)

Возвращает аннотации, связанные с текущим элементом. Если с текущим элементом не
связаны аннотации, возвращаемое значение представляет собой массив длины 0. Разница
между этим методом и *.getAnnotation(Class) заключается в том, что этот метод
определяет, является ли его аргумент повторяющимся типом аннотации (JLS 9.6),
и если это так, то пытается найти одну или несколько аннотаций этого типа, «просматривая»
аннотацию контейнера.

Вызывающий этот метод может изменять возвращаемый массив; это не повлияет на массивы,
возвращаемые другим вызывающим объектам.

- Требования к реализации:
Реализация по умолчанию сначала вызывает *.getDeclaredAnnotationsByType(Class), передавая
annotationClass в качестве аргумента, далее :
- Если возвращаемый массив имеет длину больше нуля, массив возвращается.
- Если возвращаемый массив имеет нулевую длину, а этот AnnotatedElement является классом,
  а тип аргумента является наследуемым типом аннотации, и суперкласс этого AnnotatedElement
  не равен нулю, то возвращаемый результат является результатом вызова *.getAnnotationsByType(Class)
  для суперкласса с annotationClass в качестве аргумента. В противном случае возвращается массив
  нулевой длины.

- Параметры типа: T - тип аннотации для запроса и возврата, если он присутствует.
- Параметры: annotationClass - объект Class, соответствующий типу аннотации.
- Возвращает: все аннотации текущего элемента для указанного типа аннотации, если они связаны с
              текущим элементом, иначе получаем массив нулевой длины.
- Ошибки: NullPointerException - если данный класс аннотации имеет значение null.
