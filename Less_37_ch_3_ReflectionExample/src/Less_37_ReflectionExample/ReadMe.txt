Задача:
- Имеем класс имитирующий простой калькулятор,
который реализует сложение, вычитание, деление и сумму.
- Имеется текстовый файл с тремя строками, первая строка
соответствует названию одного из методов калькулятора,
две другие строки соответствуют аргументам передаваемым
в любой из методов калькулятора.
- Необходимо написать код, который считывает данные из
текстового файла и по средствам рефлексии определяет,
что за метод соответствует первой строке текстового
файла, а далее проводит математическую операцию согласно
использованного метода, класса калькулятора.
